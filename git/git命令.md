# git常用命令

## 拉取代码
git pull

## 克隆
git clone xxx

## 创建分支
git checkout -b xxx

## 切换分支
git checkout xxx

## 删除分支
git branch -d xxx

## 删除远程分支
git push origin -d xxx

## 提交到本地仓库
git commit -m xxx

## 推送到远程仓库
git push

## 查看不同
git diff

## 添加修改
git add file_name

## 删除文件
git rm file_name

##删除文件夹 
git rm -r --cached folder_name
